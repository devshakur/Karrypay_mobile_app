# KarryPay - Mobile Financial Services App

![KarryPay Banner](./assets/images/banner.png)  

**KarryPay** is a comprehensive mobile fintech application built with **React Native** and **Expo**. It serves as an **all-in-one digital wallet and payment platform** for secure, fast, and convenient financial services.

## Features

- **Digital Wallet & Money Transfer**
  - Send and receive money quickly
  - Check wallet balance with a secure eye toggle
  - Fast transfers between users

- **Biometric Security**
  - Africa's first biometric payment system
  - Face authentication for payments
  - Fingerprint authentication support
  - Multiple authentication methods for enhanced security

- **Bill Payments**
  - Electricity, water, and cable TV subscriptions
  - Airtime top-ups & mobile data purchases
  - Education/school fee payments

- **Virtual Card Management (KarryCard)**
  - Secure virtual cards for online payments
  - Works globally for online transactions
  - Instant card activation
  - Safe and encrypted for subscriptions

- **Savings Features**
  - Flexible savings options
  - Track and manage your savings goals
  - Automatic interest accumulation
